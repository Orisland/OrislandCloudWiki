# 安装游戏

## 软件准备

如果您的电脑没有安装过Java，您可以跳转到Java安装章节 [java-an-zhuang.md](../java-an-zhuang.md "mention")

请先下载HMCL [#hmcl](../qi-dong-qi.md#hmcl "mention")。

在您的电脑上找一个位置用来存放mc，请使用空白文件夹单独放置启动器。将下载的HMCL启动器放在这里。
